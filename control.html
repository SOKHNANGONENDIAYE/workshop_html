Presentation d'une balise sémantique

Qu'est-ce qu'une balise sémantique ? Jusqu'à présent, nous nous sommes concentrés sur l'utilisation de HTML pour structurer nos pages Web et fournir une présentation claire du contenu. Suivant la norme HTML5, vous devez indiquer clairement la signification de chaque élément du contenu de votre page Web, c'est ce que nous appelons le HTML sémantique. Se concentrer sur l'utilisation de HTML pour structurer sémantiquement votre contenu Web vous offre plusieurs avantages : • Rendre votre contenu Web beaucoup plus accessible aux lecteurs handicapés. • L'application de style CSS devient plus cohérente et prévisible. • Moteurs de rechercheoptimisant et comprenant mieux vos pages Web. Les balises sémantiques : Comme nous l'avons déjà mentionné, les balises sémantiques sont utilisées pour ajouter du sens à notre document HTML. Un code doit être compréhensible par quiconque le lit. Notre document est généralement divisé en trois sections différentes : < en-tête > L'en-tête est généralement un groupe d'aides d'introduction ou de navigation. Il peut contenir certains éléments comme un logo, un formulaire de recherche, un slogan, etc. < principal > La section principale sera expliquée plus en détail dans les diapositives suivantes. < pied de page > Le pied de page contient généralement des informations sur l'auteur, des informations de contact, des données de copyright, etc.
